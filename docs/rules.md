# ゲームルール

## 盤面

- 立方体の6面。各面は 9×9 のマス（合計 6 × 81 = 486マス）
- 面の識別子：`U`(上) / `D`(下) / `F`(前) / `B`(後) / `L`(左) / `R`(右)

## 1. 各面の数独ルール（通常の数独と同じ）

各面について、以下を満たす：

- 各**行**（9マス）に 1〜9 が1つずつ
- 各**列**（9マス）に 1〜9 が1つずつ
- 各**3×3ブロック**（9個）に 1〜9 が1つずつ

## 2. 辺の共有（Sudocube 固有）

隣り合う2つの面は、接する**辺の9マス**を共有する。立方体を組み立てたときに物理的にくっつく辺なので、その2マスは**常に同じ数字**でなければならない。

- 立方体には辺が12本ある
- 各辺で、2つの面の端の列（9マス）が1対1で対応し、対応するマスは同値
- 向き（どちらの端がどちらに対応するか、反転するか）は立方体の幾何で機械的に決まる（[geometry.md](geometry.md) 参照）

## 3. 頂点の共有

立方体の**頂点（角）では3面が1点に集まる**。この角のマスは3面すべてで共有され、3つのマスが同じ数字になる。

- 立方体には頂点が8個ある
- 各頂点で3面の角マスが同値

## 通常の数独との違い（まとめ）

| | 通常の数独 | Sudocube |
|---|---|---|
| 盤面 | 9×9 が1枚 | 9×9 が6面 |
| 面をまたぐ制約 | なし | 辺（12本）・頂点（8個）で共有 |
| 1マスが属する制約 | 行・列・ブロック | それ + 共有相手の面の行・列・ブロック |

辺・頂点で共有されるマスは、自分の面のルールと、繋がった相手の面のルールの**両方**を同時に満たす必要がある。これが Sudocube の難しさと面白さの源。
