# アプリ概要

## コンセプト

Sudocube は、ふつうの数独を立方体の6面に拡張したパズル。

平面の数独が1枚の9×9盤面なのに対し、Sudocube は立方体の各面に9×9盤面を持つ。ただし6枚がバラバラに存在するのではなく、**隣り合う面が辺で数字を共有する**ことで、6面が1つの繋がったパズルになる。立方体を折りたたんだときに物理的にくっつく辺のマスは、常に同じ数字を持つ。

これにより、ある1面の数字を決めると、その情報が辺を越えて隣の面のロジックにまで波及する。1枚の数独を6枚並べただけの寄せ集めではなく、6面が相互に拘束し合う一体のパズルになっているのが本質。

## なぜ面白いか

- **面をまたぐ推論**：辺のマスは2面、頂点のマスは3面に同時に属する。1つの数字が複数面の制約に効くため、平面数独にはない「面を越えた読み」が生まれる
- **立体の把握**：どの面とどの面がどの向きで繋がっているかを頭の中で組み立てる楽しさ
- **1本の解に収束する快感**：制約が密なぶん、少ないヒントでも解が一意に定まりうる（→ 難易度の検証は [generation.md](generation.md) 参照）

## 体験のねらい（UI）

- 立方体を**3Dで表示**し、ドラッグで自由に回転できる
- ただ滑らかに回るだけでなく、手を離すと**最寄りの面がカメラ正面にきっちりスナップ**する（離散スナップ回転）
- 正面に来た面で数字を入力する。辺のマスに入力すると、共有相手の面にも同じ数字が反映される

3D表示・回転・スナップの実装方針は [roadmap.md](roadmap.md) を参照。
